Инструкция по установке Taberna eCommerce CMS:
1) Залить базу. Дамп базы данных (MySQL) хранится в файле dump.sql
2) Настройки доступа к базе данных можно изменить в файле config.db.php
3) Установите права доступа 777 (rwxrwxrwx) для следующих папок:
cache/*.*
syscache/*.*
uploaded/*.*
4) e-mail администратора и отправителя писем (для автоматически генерируемых событий, например, заказа товара) можно отредактировать в файле config.php
config.php - все глобальные настройки системы.

Вход в админку: youdomain.ru/ru/admin/ или youdomain.ru/admin/ (в зависимости от настройки $config['lang.location_show'] в файле config.php)
логин: root
пароль: password
