Конструктор интернет-магазинов с набором сервисов для создания онлайн-бизнеса.

Условия распространения и использования в файле LICENSE.txt

Инструкция по установке:
1) Залить базу. Дамп базы данных (MySQL) хранится в файле dump.sql
2) Настройки доступа к базе данных можно править в файле config.db.php
3) Необходимы права доступа 777 (rw) на создание и изменение для следующих папок :
cached/*.*
compilled/*.*
img/*.*
4) e-mail администратора и отправителя писем (для автоматически генерируемых событий, вроди заказа товара) можно отредактировать в файле config.php
config.php - все глобальные настройки системы.

Подробная инструкция по установке: http://wiki.radcms.ru/index.php?title=Установка_системы_из_архива

Последние обновления и новости на сайте http://www.tabenaecommerce.ru
